독립된 프로젝트들이 모여있는 저장소입니다.

backend/spring/
  - Spring Boot 기반 백엔드 서버 프로젝트

backend/node/
  - Node.js 기반 백엔드 서버 프로젝트

react/
    - React.js 기반 프론트엔드 프로젝트

game-server/
  - C++ 기반 게임 서버 프로젝트

video-editor/
    - node.js + react + c++/ffmpeg 기반 동영상 편집기 프로젝트

각 프로젝트의 the-shortest-distance/ 폴더는 "모르는 것은 찾아보면서 구현한다"라는 모토를 위해, 적어도 첫 코드를 작성할 수 있도록 돕는 문서입니다.

모든 프로젝트의 the-shortest-distance/는 해당 프로젝트의 design/ 문서들을 참조하여 작성되어야합니다.

즉, video-editor/the-shortest-distance/와 video-editor/design/ 문서들을 읽어보는 것이 다른 프로젝트의 the-shortest-distance/ 문서를 작성하는 데 도움이 될 것입니다.

모든 프로젝트는 독립된 저장소이므로, 예시로 읽어본 video-editor/the-shortest-distance/ 문서의 내용이 다른 프로젝트의 the-shortest-distance/ 문서에서 필요한 내용임에도 불구하고 생략되어서는 안됩니다.
